# Web Technologies - Laboratory 05 - Exercise 01

## Exercise

### Building a TODO List REST API with CRUD Operations

- [ ] **Create** an Item:
    - Operation type: **POST**
    - Path: /todos
    - Request body: Item
    - Response: Item with generated id

- [ ] **Read** an Item:
    - Operation type: **GET**
    - Path: /todos/{id}
    - Path variable: Item id
    - Response: Item

- [ ] **Read** all Items:
    - Operation type: **GET**
    - Path: /todos
    - Response: Item with generated id

- [ ] **Update** an Item:
    - Operation type: **PUT**
    - Path: /todos/{id}
    - Path variable: Item id
    - Request body: Item
    - Response: Updated item

- [ ] **Delete** an Item:
    - Operation type: **DELETE**
    - Path: /todos/{id}
    - Request body: Item
    - Response: No content
